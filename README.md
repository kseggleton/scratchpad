# Introduction
ScratchPad is a [PopClip](https://pilotmoon.com/popclip/) extension that is designed to store snippets of text, as bullet points, in a markdown formatted file. Text that is highlighted in a document, PDF or web is copied via the PopClip extension to the document.

# Configuration

* After installing the extension navigate to Users/username/Library/Application Support/PopClip/Extensions and control click on *ScratchPad.popclipext*   
* Click on *Show package contents*  
* Open *scratchpad.sh* in a plain text editor  
* Change the path in the shell script to a path where you will store the scratch pad text file
